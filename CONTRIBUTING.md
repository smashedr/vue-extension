## Quick Start

Install dependencies:

```bash
npm i
```

Start development server:

```bash
npm run dev
```

Open Chrome and navigate to `chrome://extensions/`, enable "Developer mode",
and load the unpacked extension from the `dist` directory.

## Building

```bash
npm run build:all
```
